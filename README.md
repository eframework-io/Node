# EFramework for Node

[![DeepWiki](https://img.shields.io/badge/DeepWiki-Explore-blue)](https://deepwiki.com/eframework-io/Node)
[![License](https://img.shields.io/badge/license-MIT-blue.svg?label=License)](LICENSE)
[![Discord](https://img.shields.io/discord/1422114598835851286?label=Discord&logo=discord)](https://discord.gg/XMPx2wXSz3)

面向 Node.js 环境的开发框架，提供了一系列高效且易用的类库，专注于快速开发稳定可靠的应用。

## 功能特性

- [EFramework Luban for Node](https://github.com/eframework-io/Node.Luban)：简化了 Luban 导表工具的安装和使用。
- [EFramework Protoc for Node](https://github.com/eframework-io/Node.Protoc)：简化了 Protocol Buffers 和 gRPC 工具链的安装与使用。
